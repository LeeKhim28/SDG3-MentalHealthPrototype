  The prototype of the MindCare system demonstrates the overall user interface and interaction flow of the application. The prototype was created to simulate the interaction of the users with the system features that were identified in the system analysis and system design phases.

  The prototype includes several key functions such as user authentication, mood tracking, emotional insights, assistant chat support, therapist connection, and wind-down relaxation activities. The combination of these features helps users to keep track of their emotional health and receive essential mental health assistance when they need them.

  The following prototype screens demonstrate the key user flow within the MindCare system starting with the welcome screen and the system authentication process and then proceeds to mood tracking, emotional analysis, emotional support functionalities, and relaxation tools that ensure maximum mental health.
