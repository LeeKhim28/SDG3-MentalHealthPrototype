# User Stories

US1 – User Registration

As a new user, I want to create an account, so that I can access the MindCare system.

US2 – User Login

As a user, I want to log into my account, so that I can use the system features.

US3 – Record Mood

As a user, I want to record my mood, so that I can track my emotional condition.

US4 – View Mood Insights

As a user, I want to view my mood history, so that I can understand my emotional trends.

US5 – Chat with MindCare Assistant

As a user, I want to chat with the assistant, so that I can get emotional support and suggestions.

US6 – Connect with Therapist

As a user, I want to connect with a therapist, so that I can get professional help.

US7 – Perform Wind-Down Activities

As a user, I want to access relaxation activities, so that I can reduce stress and relax.

US8 – Join Counselling Session

As a user, I want to join a counselling session, so that I can discuss my problems with a therapist.
