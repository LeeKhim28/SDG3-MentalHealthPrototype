# User Stories

1. As a student, I want to record my daily mood so that I can understand my emotional pattern.
2. As a user, I want to take a simple mental health test so that I can check my stress level.
3. As a user, I want to view stress management tips so that I can reduce anxiety.
4. As a user, I want to book an appointment with a counselor so that I can seek professional help.
5. As a user, I want to see emergency contact information so that I can get immediate help if needed.
