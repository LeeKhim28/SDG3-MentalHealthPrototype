## Testing Objective
The objective of the user testing session was to evaluate the usability and functionality of the MindCare prototype and gather feedback from potential users.

## Participants
Number of testers: 3  
Target users: University students under 30 years old.

## Testing Tasks
Participants were asked to perform the following tasks:
1. Create an account.
2. Perform a mood check-in.
3. Chat with the MindCare assistant.
4. View wellness insights.
5. Try the Wind Down features (Breathing or Relaxing Sounds).
6. View therapist connection options.

## Observations
• Most users found the interface simple and easy to navigate.  
• The mood check-in feature was clear and easy to use.  
• Users liked the wind down features such as breathing exercise and relaxing sounds.  
• Some users expected confirmation messages after certain actions (such as requesting a therapist session).  
• Users suggested adding more chatbot responses for better emotional support.

## User Feedback
Positive Feedback:
- Clean and calming design.
- Easy navigation between features.
- Helpful wellness tools for stress relief.

Suggestions for Improvement:
- Add more chatbot interaction options.
- Provide confirmation notifications for actions.
- Include reminders for daily mood tracking.

## Improvements Planned
Based on the user testing feedback, the following improvements may be implemented in future versions:
- Add confirmation pop-up messages for important actions.
- Improve chatbot response variety.
- Add reminder notifications for daily mood check-ins.
