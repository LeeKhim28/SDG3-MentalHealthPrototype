# MindCare System – Test Cases

## Test Case 1: User Registration
Test ID: TC01  
Feature: Create Account  

Steps:
1. Open the MindCare app.
2. Click "Get Started".
3. Enter Full Name, Email Address, Password, and Confirm Password.
4. Click "Create Account".

Expected Result:
User account should be successfully created and the user is redirected to the main dashboard.

Status: Pass

## Test Case 2: Mood Check-in
Test ID: TC02  
Feature: Mood Selection  

Steps:
1. Open the Mood Check-in page.
2. Select one mood option (Happy, Stressed, Tired, Sad, or Anxious).
3. Optionally enter thoughts in the text box.
4. Click "Save Changes".

Expected Result:
The selected mood should be saved and recorded in the system.

Status: Pass
